# Ocra - A Chess Engine made from scratch

## Current Capabilities 
- Able to beat 1400 rated Stockfish (at least in terms of pieces).
- Depth-6 Perft in 7 seconds.

## Installation 

```
git clone https://github.com/devanshsonii/orca
cd Ocra/
make && ./chess_engine
```
